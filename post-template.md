## 한 줄 요약
이 글을 왜 남기는지, 나중에 다시 볼 때 어떤 점이 중요한지 짧게 적습니다.

## 공부한 것
- 이번에 새로 이해한 개념
- 직접 해보면서 확인한 내용
- 다시 봤을 때 핵심이 될 포인트

## 아직 헷갈리는 것
- 더 확인이 필요한 부분
- 다시 읽어봐야 하는 개념
- 아직 확신이 없는 판단

## 나중에 더 볼 것
- 이어서 공부할 주제
- 다음에 실험해 볼 것
- 관련된 추가 자료

## 작업 메모
- 짧지만 남겨둘 만한 생각
- 시행착오
- 실제 적용하면서 느낀 점

## 참고 링크
- [문서 또는 글 제목](https://example.com)
- [추가 참고 자료](https://example.com)

## 태그/카테고리 가이드
- `study`: 공부한 내용을 정리한 글
- `todo-study`: 앞으로 공부할 주제를 정리한 글
- `notes`: 작업 메모나 짧은 기록
